Readme file for the Pushbullet module for Drupal
---------------------------------------------

Installation:
  Installation is like with all normal drupal modules:
  extract the 'pushbullet' folder from the tar ball to the
  modules directory from your website (typically sites/all/modules).

Dependencies:
  This module has no dependencies

Configuration:
  The configuration page is at admin/config/services/pushbullet,
  where you can configure the Pushbullet module.
